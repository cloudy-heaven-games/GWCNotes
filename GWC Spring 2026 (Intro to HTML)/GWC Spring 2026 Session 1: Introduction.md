### Three Technologies Used in Websites

When you develop webpages, there are generally three types of languages you’ll eventually work with:
- **HTML**: The content (images, text, etc) on the page
- **CSS**: The styling and look of elements on the page
- **Javascript**: The action and interactive elements on the page

The minimum you need to create a website is HTML, but if you don’t have CSS or Javascript, it will be a very basic, static website with not much interaction.  We’ll start by making just a static page, with no CSS or Javascript.  Later, if there is time, we can add some style and action.

### Default Code
Once you create your empty HTML project, you’ll see the following code:

```
<!DOCTYPE html>
<html>
  <head></head>
  <body>
    <!-- Your HTML here -->
  </body>
</html>
```


You won’t see much of anything in the preview window with this code, as you can see in the screenshot below:



### Code Changes: Displaying Text

After the part of the code that says `<!-- Your HTML here – >`, try adding the following lines:

```
<h1>Section Header</h1>
<p>Here is some text</p>
```

So now your code should look like this:

```
<!DOCTYPE html>
<html>
  <head></head>
  <body>
    <!-- Your HTML here -->
    <h1>Section Header</h1>
    <p>Here is some text</p>
  </body>
</html>
```
